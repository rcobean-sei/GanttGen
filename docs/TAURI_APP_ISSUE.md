# Feature: Tauri Desktop Installer App for GanttGen

## Issue Description

Create a cross-platform desktop application using Tauri that provides users with a straightforward, dependencies-included installer for GanttGen. The app should have a clean, appealing interface that matches the styling of the Gantt charts generated by the tool.

## Requirements

### User Interface
- Clean, professional design matching GanttGen's visual style
- Uses SEI brand color palette (Reds: #F01840-#300810, Purples: #D0C8C8-#2A1C30)
- Cream background (#FFFFF8) with black header (#141018)
- System font stack for cross-platform consistency

### Core Features
1. **File Input**: Support for JSON and Excel (XLSX) input files via drag-and-drop or file picker
2. **Palette Selection**: Visual palette picker with all 8 preset options (reds, reds_b, purples_a, purples_b, purples_c, alternating, alternating_b)
3. **Preview**: Live preview of color schemes and settings
4. **Generation**: One-click Gantt chart generation
5. **Export**: HTML and PNG output with selectable destination

### Technical Requirements
- Built with Tauri v2 for smaller bundle size and native performance
- Bundles all necessary dependencies (Node.js runtime, Playwright for rendering)
- Cross-platform: Windows, macOS, Linux
- Single executable installer
