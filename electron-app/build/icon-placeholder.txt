Icon files (icon.icns, icon.ico, icon.png) should be placed here.

For now, electron-builder will use default icons if these are not present.

To create proper icons:
1. Create a 1024x1024 PNG icon
2. Use electron-builder's icon conversion tools or:
   - macOS: Use `iconutil` to convert to .icns
   - Windows: Use tools like ImageMagick to convert to .ico
   - Linux: Use the PNG directly

Recommended icon: A stylized "G" or gantt chart visualization in SEI brand colors (#F01840 red and #402848 purple)
